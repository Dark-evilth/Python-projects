# 🏦 Simple Python ATM

I built this project at **16 years old** to practice my Python fundamentals. It's a bank teller simulator that handles basic transactions.

## 🚀 Key Features
- **Secure Deposits**: Checks for negative numbers to prevent balance "cheating."
- **Safe Withdrawals**: Verifies if you have enough money before letting you withdraw.
- **Crash Protection**: Uses `try/except` blocks so the program doesn't crash if a user types letters instead of numbers.
- **Infinite Loop**: Stays open until the user chooses to "Exit."

## 🧠 What I Learned
- Managing program flow with `while True` loops.
- Error handling with `ValueError`.
- Creating clean menus with functions (`def`).
- Input validation (making sure numbers are positive).
